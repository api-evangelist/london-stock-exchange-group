# London Stock Exchange Group (LSEG)

> LSEG is a United Kingdom-based stock exchange and financial information company. Its API surface catalogued here centers on the World-Check One API - screening for KYC, sanctions, PEP, and anti-money-laundering due diligence - alongside the wider LSEG Data Platform SDK ecosystem. World-Check One API v2 is served from https://api-worldcheck.refinitiv.com/v2 (HMAC request signing); API v3 is served from https://api.risk.lseg.com/screening/v3/ (OAuth 2.0 service accounts recommended, HMAC supported).

Generated by API Evangelist from the london-stock-exchange-group repository catalog (no provider-published llms.txt was found; developers.lseg.com/llms.txt returns an HTML SPA shell).

## APIs

- [World-Check One API](https://developers.lseg.com/en/api-catalog/customer-and-third-party-screening/world-check-one-api): Screening cases, results, ongoing screening, MediaCheck adverse media, reports, groups, users, and reference data
- [World-Check One Quick Start](https://developers.lseg.com/en/api-catalog/customer-and-third-party-screening/world-check-one-api/quick-start): Credentials (OAuth 2.0 service accounts or HMAC key/secret), pilot vs production, Postman collection
- [LSEG API Catalog](https://developers.lseg.com/en/api-catalog): Full catalog of LSEG APIs (Data Platform, Real-Time, Tick History, DataScope Select, screening)
